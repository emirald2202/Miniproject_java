── lib_win — Windows JavaFX Libraries ──────────────────────────────

Place the WINDOWS version of JavaFX SDK JARs here.

Download from: https://gluonhq.com/products/javafx/
  → Select: JavaFX 21 LTS  |  Platform: Windows  |  Type: SDK

After extracting, copy these files into this folder (lib_win/):
  • javafx.base.jar
  • javafx.controls.jar
  • javafx.fxml.jar
  • javafx.graphics.jar
  • javafx.media.jar
  • javafx-swt.jar
  • javafx.web.jar
  + all .dll native files

Run the app on Windows:
  .\run.ps1

Or in VSCode:
  Run > "Run (Windows) — lib_win"
